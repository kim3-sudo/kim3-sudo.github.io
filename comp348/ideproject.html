Project Definition

A web-based platform where one can type code in, press run, and see the console output on the other side, sort of like a code playground.

Required Features

  • Must include a way to have user accounts, with specific permission levels
  • Language support: Python 3
  • Ability to upgrade the supported Python version

Optional Features

  • Ability to add additional programming languages
  • Language support: C++14 (and later, using Clang compiler)
  • Language support: C# (using Microsoft compiler)
  • Assignment, submission, and grading module
  • Safe code storage
  • Integration with Git code repositories
  • Malware/code injection mitigations

Deliverables

  • Source code
  • Binaries may be included, if needed
  • Software-specific documentation
  • List of open-source software components used
