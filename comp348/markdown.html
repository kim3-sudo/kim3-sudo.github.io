What is Markdown?

Markdown is a type of markup language (remember, markup language is still code, but it's different from a programming language). It uses the .md file extension, and you can think of it as a stripped-down version of HTML, designed to nicely render in a lot of different places. Its lack of customizability is actually a boon, since it ensures that the text will be rendered correctly for wherever you put it.

It's very common in code repositories as bundled documentation, like in Git repositories as the first README.md document that's created when you open a new repository.
