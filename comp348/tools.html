Development Tools

These are tools that I like using. Be honest with yourself about what you really need. Most people only need one or two development tools to be effective.

  • Visual Studio Code
    • Publisher: Microsoft Corporation
    • Visual Studio Code starts off as just a text editor, but by adding extensions, it can be quickly turned into a full IDE for your specific environment.
    • Support for tons of languages (with more possible with extensions)
    • Support for Git GUI OOTB
    • Powerful extensions are freely available (though like any software, you should vet the credibility of any extensions you install)
    • Writing Python code? Install the Python extension!
    • Working with MySQL or MariaDB databases? Install the MySQL Shell for VS Code extension!

Development Libraries & Frameworks

These are libraries and frameworks that I've used and can suggest you use (if they're right for your use case).

  • TinyMCE
    • Need a WYSIWYG editor? Use TinyMCE, a pure JavaScript library, to power a WYSIWYG editor rendered client-side.
    • TinyMCE also handles some of the annoying stuff, like HTML injection. It'll automatically make the tags that are created in the WYSIWYG editor and escape special characters.
    • Cost: Free (OSS) (with paid version)
  • Bootstrap 5
    • Bootstrap versions matter, and Bootstrap 5 removes the dependency on Jquery. You can still use Jquery, if you need it, though!
    • Bootstrap 5 helps you with more automatic calculations, but still lets you customize things to your needs.
    • Word of warning: don't confuse the documentation and compatible functions between the different major versions of Bootstrap!
    • Cost: Free (OSS)
  • Electron
    • If you're already familiar with web-side development and you have a need to do some desktop stuff too, Electron is a good way to do that.
    • You may need to familiarize yourself with JavaScript (as a programming language) and Node.
    • Proceed with caution: Electron apps have some notoriety for being memory and storage hogs! (Discord, Spotify, and Slack are Electron or Electron-like apps; take a look at their app sizes!)
    • Cost: Free (OSS)
  • Monaco
    • The same editor that powers Visual Studio Code
    • Cost: Free (OSS)
